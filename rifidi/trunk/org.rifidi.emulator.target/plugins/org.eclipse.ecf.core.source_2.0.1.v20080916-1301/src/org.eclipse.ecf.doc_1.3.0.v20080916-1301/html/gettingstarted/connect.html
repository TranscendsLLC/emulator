Eclipse Communication Framework (ECF)

Connecting an ECF Collaboration Client to a Test Server

  1. Make sure ECF plugins are installed in one of three ways described here.
  2. Open the Navigator View or the Package Explorer View.
  3. Select any Project (first create some new project if none previously exists).
  4. Right-click on the project to bring up the context menu for the selected project.
  5. Select the ECF menu toward the bottom of the menu and select one of:
    • 'Join Collaboration...' This will allow you to specify an URL of the group to join. You must know of a currently running server (and know it's URL) to use this option.
    • 'Join Test Collaboration (Localhost)' This will connect you directly to a test server running on localhost. See Running a Collaboration Server on Localhost if you wish to run this way.
    • 'Join Test Collaboration (Composent)' This will connect you directly to a test server running on composent.com. This is a test server setup for the ECF team usage. You are welcome to try out connecting to this server to test out/use the ECF example client. The server's availability may not be consistent, however.
