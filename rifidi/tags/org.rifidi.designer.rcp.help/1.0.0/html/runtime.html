Runtime Controls

Move Tool

w - move camera forward
a - move camera left
s - move camera backward
d - move camera right
left click - select entity
left click + mouse move - move entity around or, if a container like a clothing rack is selected, grab an item out of the container
right click - select entity and show context menu
mouse wheel - if entity is selected move the entity up/down, if no entitiy is selected zoom in/out

Watcharea Tool

w - move camera forward
a - move camera left
s - move camera backward
d - move camera right
left click and hold - start drawing watch area, on release the area is created NOTE: watch ares can only be deleted by going to designtime and rightcliking them in the list on the left.
